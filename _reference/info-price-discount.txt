================================================================================
[2] 스픽 할인코드 적용 방식 (핵심 포인트)
================================================================================
- 코드를 직접 입력하는 방식이 아님
- 전용 링크 접속 시 가입 화면에서 할인가가 자동 표시
- "코드 입력란 어디 있나요?" → 입력란 없음, 자동 적용
- 반드시 전용 링크를 경유해야 함 (앱 직접 결제·일반 웹 접속 시 할인 미적용)
- 웹(브라우저) 결제만 할인 적용 (앱스토어 결제는 수수료로 가격 다름)
- 추천인 링크 기반 = 친구 추천 할인과 동일 메커니즘, 중복 적용은 안 됨


================================================================================
[3] 무이자 할부
================================================================================
- 12개월 무이자 할부 가능
- 할부 결제도 할인 동일 적용 (할인 금액 안 줄어듦)
- 결제 화면에서는 할부 선택이 어려울 수 있음 → 카드사 앱에서 사전 설정 권장
  (카드사 앱 → 결제·이용 메뉴 → 무이자 할부 신청)


================================================================================
[4] 할인 안 될 때 체크포인트
================================================================================
- 가입 후 7일 이내 결제해야 할인 유지
- 기존 가입 이력 있으면 새 이메일로 재가입
- 앱 결제 말고 웹 결제로 진행
- 브라우저 캐시 문제 → 시크릿 모드로 재접속
- 한국에서 원화(KRW) 결제 (해외/타지역은 환율 차이)


================================================================================
[5] 프리미엄 vs 프리미엄 플러스 (가격 차이 설명용)
================================================================================
- 가격 차이: 연 13만 원 (하루 약 356원)
- 월 환산: 프리미엄 9,083원 / 플러스 19,917원 (차이 약 1만 원)
- 프리미엄: 하루 평균 10분, 습관 형성, 가성비 중심
- 프리미엄 플러스: 하루 평균 30분, AI 무제한 대화, 맞춤 레슨 무제한, 30일 챌린지
- 업그레이드 가능 / 다운그레이드 불가


================================================================================
[6] 환불 (가격 글에 같이 넣으면 좋은 정보)
================================================================================
- 7일 이내 100% 전액 환불
- 무료체험은 없지만, 7일 환불 제도로 사실상 체험 가능
- 7일 이후: 부분 환불 (이용분 + 위약금 10% 공제)
