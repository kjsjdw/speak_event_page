<h2 id="toc-1" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">1. 스픽 시작하기 — 회원가입과 첫 설정</h2>

<p>스픽 학습은 회원가입과 몇 가지 초기 설정을 마치면 바로 시작할 수 있습니다. 가입 자체는 5분이 채 걸리지 않을 정도로 간단합니다.</p>

<h3 style="margin-top: 30px;">🌐 웹사이트에서 회원가입하기</h3>

1. **speak.com**에 접속합니다.
2. '**시작하기**' 버튼을 클릭합니다.
3. '**계속하기**' 버튼을 눌러 가입을 진행합니다.
4. 회원가입이 완료되면, 앱스토어 또는 구글 플레이스토어에서 **스픽 앱을 다운로드**합니다.
5. 회원가입한 계정 정보로 앱에 로그인합니다.

<h3 style="margin-top: 30px;">📱 앱에서 직접 회원가입하기</h3>

1. 앱스토어 또는 구글 플레이스토어에서 **스픽 앱을 다운로드**합니다.
2. 앱을 열고 '**시작하기**' 버튼을 클릭합니다.
3. 화면에 나오는 질문에 답한 뒤 '**계속하기**' 버튼을 누릅니다.
4. 안내에 따라 회원가입을 진행합니다.
5. 회원가입한 계정 정보로 로그인합니다.

<h3 style="margin-top: 30px;">🎯 가입 시 학습 목적 선택</h3>

<p>가입 과정에서 본인의 학습 목적을 선택할 수 있습니다. 선택한 목적에 따라 추천 코스와 학습 흐름이 자동으로 조정됩니다.</p>

* 해외여행·유학
* 외국인과 프리토킹
* 이직·커리어 발전
* 자기계발
* 우리 아이와 영어로 대화하기
* 기타

<h3 style="margin-top: 30px;">🔠 모국어와 배우는 언어 설정</h3>

<p>스픽은 한국어 외에도 14개 모국어로 영어를 배울 수 있고, 영어로 5개 언어(스페인어·프랑스어·일본어·한국어·이탈리아어)를 배우는 코스도 있습니다.</p>

<p><strong>변경 방법</strong>: 스픽 앱 → 👤 프로필 → ⚙️ 설정 → <strong>코스 언어 설정</strong> → "나의 모국어"와 "배우고자 하는 언어" 선택</p>

<hr style="border: 0; border-top: 1px solid #eee; margin: 20px 0;">

<h2 id="toc-2" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">2. 영어 레벨 확인 — 스픽 레벨 시스템</h2>

<p>가입 후 학습을 시작하면 자연스럽게 본인의 스픽 레벨이 매겨집니다. 단순히 레슨을 몇 개 끝냈는지가 아니라, "<strong>영어로 실제로 무엇을 말할 수 있는지</strong>"를 기준으로 한 행동 중심 지표입니다.</p>

<h3 style="margin-top: 30px;">🎯 실력 체크 레슨으로 진단</h3>

<p>각 유닛의 마지막에는 <strong>실력 체크 레슨</strong>이 제공됩니다. 자연스러운 대화 형식으로 진행되며, 완료하시면 다음 내용을 확인할 수 있습니다.</p>

* 내가 달성한 스킬 목록
* 현재 학습 중인 스픽 레벨
* 다음 단계로 가기 위해 필요한 스킬

<h3 style="margin-top: 30px;">🗣️ 스킬(말하기 능력 문장)이란</h3>

<p>스킬은 "<strong>나는 일상적인 루틴에 대해 간단한 질문을 할 수 있어요</strong>" 같은 구체적인 말하기 능력 문장입니다. 한 레벨 안에는 여러 스킬이 포함되어 있고, 스킬을 하나씩 달성할수록 레벨이 올라가는 구조입니다.</p>

<h3 style="margin-top: 30px;">📈 스픽 레벨 허브에서 진행 상황 보기</h3>

<p>현재 본인의 레벨과 달성한 스킬, 다음 목표 스킬을 한곳에서 확인하실 수 있습니다.</p>

* **접근 위치**: 유닛 요약 화면 또는 프로필 탭
* 전체 스픽 레벨과 각 레벨의 스킬 목록을 한눈에 볼 수 있습니다.

<h3 style="margin-top: 30px;">🌐 CEFR과의 관계</h3>

<p>스픽 레벨은 국제 언어 능력 평가 기준인 <strong>CEFR</strong>(유럽공통참조기준)과 GSE를 참고해 만들어졌습니다. 다만 CEFR보다 더 구체적이고 행동 중심이라 "지금 내가 영어로 무엇을 할 수 있는지"를 더 명확하게 보여줍니다.</p>

* 초중급 (스픽 레벨 32~39)
* 중급 (스픽 레벨 40~49) — **CEFR B1**에 해당
* 중상급 (스픽 레벨 50~59)

<h2 id="toc-3" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">3. 코스 선택과 변경</h2>

<p>가입 후 자동 추천되는 코스 외에도, 직접 코스 페이지에서 본인 수준과 관심사에 맞는 코스를 골라 학습할 수 있습니다. 코스 선택 페이지는 2025년 말 업데이트되어 더 깔끔하게 정돈되었습니다.</p>

<h3 style="margin-top: 30px;">📈 레벨별 섹션</h3>

* 왕초보부터 마스터 레벨까지 **CEFR 레벨별**로 정리되어 있습니다.
* 페이지 상단의 쉬운 코스부터 아래로 내려갈수록 난이도가 올라갑니다.
* 각 코스 오른쪽에 **CEFR 레벨 표시**가 있고, 코스 설명도 함께 있어 선택 시 참고할 수 있습니다.

<h3 style="margin-top: 30px;">🎡 토픽별 선택 코스</h3>

<p>여행·비즈니스 등 <strong>상황별 주제</strong>로 구성된 코스입니다. 본인 관심사·목적에 맞춰 빠르게 골라 학습할 수 있고, 각 코스에도 CEFR 레벨이 표기되어 있습니다.</p>

<h3 style="margin-top: 30px;">🗃️ 이전 버전 커리큘럼</h3>

<p>이전에 사용하던 왕초보·레벨별 코스는 코스 페이지 <strong>맨 아래</strong>에 남아 있습니다. 수강은 가능하지만 정기 업데이트는 제한적이므로, 상단의 새 코스 이용이 권장됩니다.</p>

<h3 style="margin-top: 30px;">🔄 코스 변경 방법</h3>

<p>코스 페이지에서 원하는 코스의 "<strong>선택</strong>" 버튼을 누르면, 홈 탭에서 해당 코스의 수업이 바로 진행됩니다. 수업을 듣다가 다른 코스로 자유롭게 변경할 수도 있습니다.</p>

<h2 id="toc-4" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">4. 새로워진 홈 화면 보는 법</h2>

<p>2025년 말 업데이트로 홈 화면이 새롭게 바뀌었습니다. "<strong>지금 무엇을 학습할지</strong>"가 한눈에 보이도록 설계되어 있고, 각 학습 항목이 <strong>말풍선(스피킹 스킬)</strong> 형태로 표시됩니다.</p>

<h3 style="margin-top: 30px;">🌀 말풍선 상태로 진행 상황 확인</h3>

* **🌀 파란색으로 깜빡이는 말풍선**: 지금 학습할 차례의 레슨
* **🔵 파란색으로 채워진 말풍선**: 학습이 진행 중인 스킬
* **✅ 체크 표시**: 해당 말풍선의 모든 레슨 완료
* **📙 주황색 플래시카드 아이콘**: 1분짜리 빠른 비회화 레슨 (바쁠 때 활용)

<h3 style="margin-top: 30px;">📱 말풍선 클릭 시 동작</h3>

<p>말풍선을 누르면 해당 스킬의 레슨 카드가 펼쳐지고, 원하는 레슨을 골라 시작할 수 있습니다. 레슨을 건너뛰거나 이미 끝낸 레슨을 다시 복습할 수도 있습니다.</p>

<h3 style="margin-top: 30px;">🎯 연습 탭으로 이동된 기능</h3>

<p>홈 화면에서 <strong>왼쪽으로 스와이프</strong>하면 새로운 <strong>연습 탭</strong>이 나타납니다. 다음 기능이 이곳으로 옮겨졌습니다.</p>

* 맞춤 레슨
* 점프인 레슨 (간단한 퀴즈 형식)

<h2 id="toc-5" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">5. 레슨 종류와 진행 방식</h2>

<p>스픽에는 여러 종류의 레슨이 준비되어 있고, 각각의 진행 방식이 조금씩 다릅니다. 어떤 레슨이 어떤 역할을 하는지 알아두시면 효율적으로 학습하실 수 있습니다.</p>

<h3 style="margin-top: 30px;">🎬 코스 레슨 — 3단계 진행</h3>

<p>원어민 비디오 강의 + 직접 말하기 + AI 롤플레이로 이어지는 가장 기본적인 레슨입니다.</p>

1. **오늘의 수업**: 원어민 강사가 짧은 영상으로 핵심 표현 설명
2. **스피킹 연습**: 표현을 여러 문장으로 바꿔가며 직접 발화 (한 수업에 약 100문장 발화)
3. **실전 대화**: 상황극 형태로 AI와 롤플레이

<h3 style="margin-top: 30px;">🪄 맞춤 레슨 — AI가 자동 생성</h3>

<p>학습자의 실수와 관심사를 기반으로 AI가 자동으로 만들어 주는 레슨입니다. 부족한 부분만 모아서 보강할 수 있어 효율적이며, 프리미엄은 하루 최대 3개, 프리미엄 플러스는 무제한으로 이용 가능합니다.</p>

<p>📌 자세한 종류와 동작 방식은 <a href="/curriculum/">스픽 커리큘럼 안내</a> 페이지에서 확인하실 수 있습니다.</p>

<h3 style="margin-top: 30px;">🔵 튜터 레슨 — 1:1 대화형 코스 수업</h3>

<p>비디오 형식이 아니라 AI 튜터와 <strong>1:1로 대화하면서 진행되는 수업</strong>입니다. 학습 속도를 스스로 조절할 수 있어 보다 능동적인 학습이 가능합니다.</p>

<h2 id="toc-6" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">6. 꾸준히 학습하게 만드는 시스템</h2>

<p>아무리 좋은 커리큘럼도 꾸준히 쓰지 않으면 효과가 약합니다. 스픽은 학습을 멈추지 않도록 돕는 여러 동기 부여 시스템을 함께 제공합니다.</p>

<h3 style="margin-top: 30px;">🔥 불꽃 — 연속 학습 기록 시스템</h3>

* 매일 1개 이상 레슨을 완료하면 그날의 불꽃이 켜집니다.
* 누적 일수가 길어질수록 만족감과 동기부여가 커집니다.
* 하루를 놓치셨더라도 24~48시간 이내에 레슨 2개를 완료하시면 **불꽃 되살리기**가 가능합니다.

<h3 style="margin-top: 30px;">🏆 헌드레드 클럽 — 100일 달성 한정 혜택</h3>

<p>100일 연속 불꽃을 달성한 분에게만 <strong>전용 티셔츠 신청권</strong>이 제공됩니다. 한 달 채우기도 쉽지 않은 영어 공부를 100일 동안 이어 오신 분들에게 주는 스픽의 작은 선물입니다.</p>

<h3 style="margin-top: 30px;">🏅 리그 — 매주 다른 학습자와의 친선 경쟁</h3>

* 매주 비슷한 수준의 학습자 <strong>최대 30명</strong>으로 구성된 그룹에 자동 배정됩니다.
* 일주일간 누가 더 많은 XP를 쌓는지 경쟁합니다.
* <strong>10개의 티어</strong>(아이언 → 챌린저)가 있고, 매주 결과에 따라 승급·유지·강등이 결정됩니다.
* 경쟁이 부담스러우시면 **프로필 → 설정 → 환경설정**에서 리그 기능을 끄실 수도 있습니다.

<h2 id="toc-7" style="padding-top: 100px; margin-top: -80px; padding-bottom: 25px;">7. 상황별 학습 도구</h2>

<p>지하철·도서관·소음이 신경 쓰이는 환경 등 상황에 따라 학습을 멈추지 않고 이어갈 수 있도록, 스픽은 여러 보조 모드를 제공합니다.</p>

<h3 style="margin-top: 30px;">💬 비주얼 모드 프리톡 — 3D 캐릭터와 실시간 대화</h3>

<p>텍스트 카드 위주의 기존 프리톡과 달리, <strong>3D 캐릭터와 음성 중심으로 실시간 대화</strong>를 주고받는 새로운 형식입니다. 실제 사람과 대화하는 듯한 느낌을 줍니다.</p>

<div class="step-media" style="margin: 20px 0;">
  <video autoplay loop muted playsinline preload="metadata" style="width: 100%;">
    <source src="/content/videos/how-to-use-tutor.mp4" type="video/mp4">

* '나만의 시나리오 만들기'에서 캐릭터와 상황을 직접 선택할 수 있습니다.
* 대화 도중 언제든 끼어들기가 가능합니다.
* 사용 방법: 프리톡 레슨 시작 전 화면에서 **비주얼 모드** 선택

<h3 style="margin-top: 30px;">🎙️ 핸즈프리 모드</h3>

<p>화면을 매번 터치하지 않아도 수업이 자동으로 진행됩니다. 운전 중·요리 중처럼 손이 자유롭지 않은 상황에서 유용합니다.</p>

* **사용 방법**: 레슨 화면 오른쪽 위의 **핸즈프리 버튼** 활성화

<h3 style="margin-top: 30px;">🤫 말할 수 없어요 모드</h3>

<p>소리 내어 말하기 어려운 공공장소에서, 스피킹 부분을 <strong>스픽의 모범 발음이 대신 말해주는</strong> 기능입니다. 오늘의 수업에서 제공됩니다.</p>

* **사용 방법**: 오늘의 수업 우측 상단의 **'말할 수 없어요' 버튼** 활성화

<h3 style="margin-top: 30px;">📚 문장·단어 저장으로 복습 자료 만들기</h3>

<p>수업·프리톡·스픽 튜터 대화에서 좋다고 느낀 표현은 북마크로 저장해 두실 수 있습니다.</p>

* **수업 중 저장**: 스피킹 카드 오른쪽 위의 책갈피(북마크) 아이콘
* **프리톡 저장**: 프리톡 탭 → 시계 아이콘으로 학습 기록 열기 → 책갈피 아이콘
* **스픽 튜터 저장**: 튜터 메시지를 길게 눌러 '저장하기' 선택
* **저장 위치**: 프로필 → 보관한 단어/표현 (스픽 튜터에서 저장한 항목은 '튜터' 탭에서 별도 확인)
